URL Entry: The user types a web address (URL) into the browser.

DNS Resolution: The browser contacts a DNS server to find the IP address of the web server hosting the site.

HTTP/HTTPS Request: The browser sends an HTTP (or HTTPS) request to the server for the desired web page (usually an HTML file).

Server Response: The web server locates the requested file and sends it back to the browser along with a status code (e.g., 200 OK).

Rendering:

The browser parses the HTML file.

It requests linked resources such as CSS, JavaScript, and images.

The CSS styles the content.

The JavaScript adds interactivity and dynamic behavior.

Display: The browser renders the final, styled web page on the screen
